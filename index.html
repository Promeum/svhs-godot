Eclipse II: The Darkest Day

By Jayden Williams

Eclipse is a platformer game in which speed is of the essence, pass through obstacles, enemies, and the in-game guide (?! ) to complete levels. In this parkour platformer game you will need to avoid ducks and bats to escape the water! Each map has 3 stars, collect stars around the map and watch as the progress builds up in the level select. Be wary though, each replay does cause a reset in collected stars in the selected map.

Find The Sauce

By Xavier Gray and David Amewu

Find The Sauce is an exciting single-player, decision-based game, originally developed as part of a submission for the SkillsUSA competition. Each decision impacts the storyline, leading to multiple possible outcomes that challenge your critical thinking and problem-solving skills.
